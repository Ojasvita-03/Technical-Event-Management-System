# Technical Event Management System

This is a Python-based menu-driven application developed using Google Colab.

## Features
- Admin Signup & Login
- Vendor Signup & Login
- User Signup & Login
- Product Management
- Cart & Checkout
- Order Tracking

## Technology Used
- Python
- Pandas
- Google Colab

## How to Run
1. Open the notebook in Google Colab
2. Run all cells
3. Follow the menu options in output
